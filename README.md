🚗 PoolGo

Smart AI-Powered Carpooling for Everyday Travel

PoolGo is an AI-powered carpooling mobile application designed to make daily commuting smarter, cheaper, and more efficient. The platform connects people traveling on similar routes, helping users share rides, reduce travel costs, minimize traffic congestion, and support eco-friendly transportation.

Built especially for urban commuters like students, IT employees, and job seekers, PoolGo uses intelligent ride matching and real-time navigation to provide a seamless shared travel experience.

✨ Features
🤖 AI-based smart ride matching
📍 Real-time Google Maps navigation & live tracking
🚗 Ride request & acceptance system
🔐 OTP-based secure ride start
⏱️ ETA and traffic-aware route optimization
💰 AI-powered fare calculation
👥 Nearby ride giver & vehicle detection
📱 Rider & Driver modes
💳 Wallet & digital payment integration
🛡️ Safety features including SOS & trip sharing
📜 Ride history & booking management
🎁 Referral & rewards system
